# CareerPilot AI

CareerPilot AI is a Multi-Agent Career Guidance System built using Google ADK 2.3 and Gemini 2.5 Flash.

## Features

- Career Agent
- Skills Agent
- Resume Agent
- Interview Agent
- Roadmap Agent
- Skill Gap Analyzer Tool

## Technologies

- Python 3.11
- Google ADK 2.3
- Gemini 2.5 Flash

## Run

1. Install requirements
2. Create a `.env` file with your Google API key
3. Run:

```bash
adk web
